PT MULOH GLOBAL TRADE WEBSITE
----------------------------------
Terima kasih telah menggunakan website ini.

CARA MEMBUKA WEBSITE:
1. Ekstrak file ZIP ini di ponsel atau komputer Anda.
2. Buka file bernama "index.html" menggunakan browser (Chrome, Safari, dll).

CARA MENGGANTI KONTAK / NOMOR WHATSAPP:
1. Buka file index.html dengan aplikasi teks (misalnya QuickEdit / Notepad).
2. Cari teks: 6281234567890
3. Ganti dengan nomor WhatsApp bisnis Anda (gunakan format internasional tanpa tanda +, contoh: 6289876543210).
4. Simpan dan buka ulang file index.html di browser.

CARA MENGGANTI LOGO:
1. Ganti file "logo.svg" dengan logo Anda yang baru.
2. Pastikan nama filenya tetap "logo.svg".

Website ini sudah bilingual (Indonesia & English) dan responsif untuk tampilan Android.
